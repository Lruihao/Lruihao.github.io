前面我有文章提到怎么提交本地文件到github,coding等远程仓库。每次可以分为三个步骤

  • git add * (添加需要提交的文件，这里全选)
  • git commit -m “提交信息”
  • git push

但是这样感觉很麻烦每次都要重复输入提交命令和提示信息。
